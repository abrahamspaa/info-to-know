# Express creation 

#### Step 1 
Step up a new node application
```
npm init -y 
```
#### Step 2
Added express and dependencies
```
yarn add body-parser cors express helmet jsonwebtoken mongoose morgan
```
### Step 3
Created index file 
```
mkdir source && touch source/index.js
```
