# ubuntu

## How to install 
`sudo apt-get install nginx`

## To See the status 
`systemctl status nginx`

## To see in the browser 
`http:localhost/`

## To stop 
`sudo systemctl stop nginx`

## To start 
`sudo systemctl start nginx`

## To restart 
`sudo systemctl reload nginx`
