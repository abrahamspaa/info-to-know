# ubuntu

## How to install 
`sudo apt install -y mongodb`

## To See the status 
`systemctl status nginx`

## To see in the browser 
`http:localhost/`

## To stop 
`sudo systemctl stop mongodb`

## To start 
`sudo systemctl start mongodb`

## To restart 
`sudo systemctl reload mongodb`
